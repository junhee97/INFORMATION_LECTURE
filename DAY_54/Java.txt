람다함수
	익명 함수를 지칭하는 용어
	수학에서 사용하는 함수를 보다 단순하게 표현하는 방법

형태
	( ) -> { }
	( ) -> 1 / 함수몸체가 단일 실행문이면 { } 생략 가능
	( ) -> { return 1; } / 함수몸체가 return 문으로만 구성되어잇으면 { } 생략 가능


STREAM
	자바 스트림은 자바 8부터 도입된 기능으로,
	컬렉션을 처리하고 변환하기 위한 강력한 기능을 제공합니다
	스트림을 사용하면 데이터를 효율적으로 처리하고 다양한 연산을 수행할 수 있습니다

함수 종류

중개 연산 ( Intermediate Operations )
	스트림의 요소를 변환하거나 필터링 하는 작업을 수행, 파이프라인에 의해 여러번 호출 가능
	중개 연산만으로는 실제로 데이터가 처리되지 않으며, 최종 연산이 호출될 때까지 연산이 지연
	★ filter - 주어진 조건에 맞는 요소를 선택
	★ map - 요소를 다른 형태로 변환
	flatMap - 각 요소를 변환하고, 이를 하나의 스트림으로 평면화
	distinct - 중복된 요소를 제거
	sorted - 요소 정렬
	limit - 주어진 개수 만큼 요소를 제한
	skip - 처음 몇 개의 요소를 제외하고 반환

최종 연산 ( Terminal Operations )
	★ forEach - 각 요소를 반복적으로 처리
	★ collect - 요소를 수집하여 새로운 컬렉션을 생성
	★ reduce - 요소를 하나로 줄여서 결과를 반환
	min - 최소값 반환
	max - 최대값 반환
	count - 요소의 개수 반환
	anyMatch - 하나 이상의 요소가 주어진 조건과 일치하는지 검사
	allMatch - 모든 요소가 주어진 조건과 일치하는지 검사
	noneMatch - 모든 요소가 주어진 조건과 일치하지 않는지 검사
	findFirst - 첫번째 요소 반환
	findAny - 임의의 요소 반환


함수형 프로그래밍
	거의 모든 것을 순수 함수로 나누어 문제를 해결하는 기법으로,
	작은 문제를 해결하기 위한 함수를 작성하여 가독성을 높이고 유지보수를 용히하게 해준다

@FunctionalInterface
	함수형 인터페이스 선언 (Compiler 에게 전달) - 추상메소드가 하나만 존재
