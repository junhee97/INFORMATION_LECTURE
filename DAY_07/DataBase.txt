데이터베이스 구축 절차


1. 요구사항 분석
2. 설계
	개념적 설계 : 도식화
	논리적 설계 : 구체화
	물리적 설계 : 구현
3. DB 구축(구현)
4. DB 튜닝(인덱스, 뷰, ...)
5. 응용프로그램과의 연결(웹, 앱)



개념적 설계
	개체관계모델(ERD)

	개체 : 사각형
	속성 : 동그라미
	관계 : 마름모


	관계의 종류
	1 : 1 - 관계되어 있는 두 캐체 타입이 모두 하나씩 개체 어퍼런스를 갖는 관계
	1 : N - 한 개체 타입은 여러개의 개체 어퍼런스를 갖고 다른 한쪽은 하나의 개체 어퍼런스를 갖는 관계
	N : M - 두 개체 타입 모두 여러개의 개체 어퍼런스를 갖는 관계
