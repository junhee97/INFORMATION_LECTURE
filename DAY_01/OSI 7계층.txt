1. Network 는 무엇이며 무엇을 목적으로 하는가?
   데이터 교환을 위해 통신장치를 연결한 망

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
   LAN / WAN

3. LAN 의 특징에 대해서 서술하세요
   근거리 통신망
   비교적 작은 범위 내의 컴퓨터와 네트워크 장치들을 연결하여 데이터를 주고받을 수 있도록 하는 네트워크

4. WAN의 특징에 대해서 서술하세요
   원거리 통신망
   넓은 범위에 걸쳐 있는 네트워크로 서로 떨어져 있는 지역을 연결

5. InterNet 이란
   전 세계적으로 연결된 컴퓨터 네트워크의 집합

6. 프로토콜은 무엇인가?
   네트워크에서 데이터 통신을 위한 규칙
   3요소 : 구문, 의미, 타이밍

7. 네트워킹 모델의 목적은 무엇인가?
   다양한 기기 간 통신을 위해 약속된 구조
   통신만 구현을 위한 지침서
   하자 발생 시 참고하는 지침서
   이기종간 장치간 통신을 위해 미리 정의해 놓은 표준 프로토콜 모델

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
  OSI 7계층
	애플리케이션 계층
	프레젠테이션 계층
	세션 계층
	전송 계층 : 송수신간의 데이터의 신뢰성을 확보
	네트워크 계층 : 데이터를 목적지까지 최적의 경로로 전달하는 계층
	데이터링크 계층 : 두 장치간에 데이터를 주고 받을 수 있도록 하는 계층(포인트 투 포인트) 
	물리 계층 : 장치 간 물리적인 연결


1. 물리계층 하는일/역할/관련 장치는?
   실제 장치들을 연결하기 위해 필요한 세부 사항들의 정의
   데이터 전달
   케이블, 허브, 리피터

2. Cable 이란
   네트워크 장비나 장치를 연결하는 데 사용되는 물리적인 매체

3. LAN Cable 종류
   이더넷 케이블, 광섬유 케이블, 동축 케이블

4. 부호(Code) 란?
   의미를 가지는 약속된 기호

5. 신호(Signal)이란?
   역할 : 원격지로의 부호 전달
   약속 : 
   장치 : 케이블, 리피터, 허브

6. 신호전달에 필요한 요소?
   전기에너지, 통신망

7. 리피터란?
   신호 증폭 장치

8. 허브란?
   여러 대의 컴퓨터 간 통신 가능
