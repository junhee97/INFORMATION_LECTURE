Java 언어 : 절차지향 문법 ( C ) + 객체지향 문법 ( C++ )

package - 폴더 개념 ( 패키지 구별 코드 )

public class { } - 객체지향 문법이 적용되는 범위 ( 클래스 영역 )

public static void main(Stirng[] args) { } - 절차지향 문볍이 적용되는 범위 ( 메소드 영역 )


메소드 종류
	라이브러리 메소드 : 미리 만들어져 제공되는 메소드
	사용자정의 메소드 : 개발자에 의해 만들어지는 메소드
	main 메소드 : 최초 실행 메소드


System.out.print( );
	\n : 개행, 줄바꿈
	\b : 백스페이스
	\t : 탭길이(기본 8칸) 만큼 커서 이동

System.out.printf( );
	format : 형식, 서식
	%d : 10진수 정수 서식문자
	%f : 10진수 실수 서식문자
	%c : 한문자 서식문자
	%s : 문자열 서식문자
	변수처리 하기 위해서, 한번에 여러 자료형의 문자를 입력하기 위해서 사용

System.out.println( );
	ln : 개행, 줄바꿈


진수
	%d : 10진수 서식문자
	%o : 8진수 서식문자 ( 0 : 8진수를 의미하는 접두사 )
	%x : 16진수 서식문자 ( 0x : 16진수를 의미하는 접두사 )

 	컴퓨터(CPU)는 구조상 던셈연산을 할 수 있다 (o)
 	컴퓨터(CPU)는 구조상 뺄셈연산을 할 수 있다 (x)
 	컴퓨터(CPU)에서 뺄셈요청시 뺄셈처리는 할 수 있다 (o)
 	보수개념을 도입해서 뺄셈처리를 수행하기 때문에 가능


변수_자료형
	Data ( 수, 자료 ) : 선 저장 / 후 처리
	변수 : 개발자의 유지보수 측면에서 바뀔 예정이 큰 수
	변수명 : 저장되어져 있는 변수 공간에 접근하기 위한 문자 형태의 주소
	자료형 : Data ( 수, 자료 ) 를 저장하기 위한 공간을 형성하는 예약어 ( + 제약조건 )


자료형
	정수
	byte : 1byte
	short : 2byte
	char : 2byte - 양수값만 지원
	int : 4byte - 기본
	long : 8byte

	실수
	float : 4byte
	double : 8byte - 기본

	단일 문자
	char : 2byte

	문자열
	Stirng : 클래스 자료형 - 동적 공간 할당
