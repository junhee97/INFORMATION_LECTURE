Swing
	JDK에 포함되어 있는 GUI개발용 라이브러리로 다양한 컴포넌트와 이벤트 처리 구현이
	가능한 클래스, 메소드를 제공합니다
	컨테이너와 그안에 붙여진 컴포넌트들로 구성

컴포넌트 ( Component )
	컨테이너에 포함되어야만 화면에 출력 될 수 있는 GUI 객체

컨테이너 ( Container )
	사용자 인터페이스 요소들
	컴포넌트들을 담고 그룹화 하거나 배치하는 역할

패널 ( Panel )
	컨테이너를 상속받은 클래스로 일반적으로 컴포넌트들을 구성하고 
	배치하기 위해 사용되는 컨테이너
