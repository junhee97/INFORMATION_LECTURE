VI : 문서 편집기

   이동모드 : 편집하기 위해 특정위치로 커서 이동
	커서 이동 - H : 왼쪽으로 이동  J : 아래로 이동  K : 위로 이동  L : 오른쪽으로 이동
	단어 이동 - E : 다음 단어의 끝으로 이동  W : 다음단어의 처음으로 이동  B : 이전단어의 처음으로 이동
	행 이동 - 0 : 행 처음으로 이동  $ : 행 마지막으로 이동
	페이지 이동 - ctrl+f : 다음 페이지  crtl+b : 이전 페이지
	마지막 처음 - gg : 문서의 처음  shift+g : 문서의 끝
	삭제 - x : 한글자씩 삭제  dd : 현재 커서의 행 삭제  숫자+dd : 현재 커서에서 숫자만큼 행 삭제
	복사/ 붙여넣기 - yy : 현재 커서의 행 복사  숫자+yy : 현재 커서에서 숫자만큼 행 복사  p : 붙여넣기
	r : 현재 커서의 한문자 변경
	
   편집모드 : 내용 편집
	a : 커서 한칸 뒤로 이동해서 편집
	i : 커서가 있는 행에서 한행 내려와서 편집
	o : 커서 위치에서 한행 내려와서 편집
	s : 현재 커서 위치의 문자 삭제후 편집

   실행모드 : 검색(/), 치환, 외부명령어 수행
	:q : 종료
	:wq : 저장 후 종료
	:q! : 강제 종료
	:set number : 행번호 출력
	:set nonumber : 행번호 삭제
