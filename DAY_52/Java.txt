	SocketServer - 클라이언트의 연결요청을 받아 세션을 형성

Thread
	하나의 프로레스에서 여러 실행단위로 나누어 처리하도록 제공되는 기술
	하나의 프로세스는 반드시 하나이상의 스레드 단위로 실행된다
	메인스레드와 작업스레드로 나뉜다

	스레드 분할
	1. Runnable Interface 스레드 분할
		1. 작업객체 생성
		2. 메인 스레드에서 분리
		3. 스레드 실행
			new Thread ( ) { run ( ) }.start( );
	2. Thread class 스레드 분할

	run ( ) 메소드에 실행할 내용 작성
