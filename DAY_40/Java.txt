(자료) 형변환
	연산시 일치하지 않는 자료형을 일치시키는 작업

	자동형변환(암시적 형변환) : 컴파일러에 의해 자동 형변환
	강제형변환(명시적 형변환) : 프로그래머에 의해 강제 형변환

	자동형변환(=)
	'변수 연산처리'시 범위가 넓은 공간에 작은값이 대입되는 경우
	'리터럴 상수 연산처리'시 리터럴 값에 따른 형벼환 여부 결정
	byte > short,char > int > long > float > double


강제 형변환
	좁은 범위공간에 큰 값을 넣으려고 하는 경우
	기본적으로 불가능하기 때문에 강제로 자료형을 바꾸어 전달한다
	데이터 손실의 염려가 있다

정수 연산식 ( int 보다 작은 변수 자료형 ( short, char, byte) )
	연산식을 수행하면 int 형으로 자동 형변환


문자 형변환
	문자열 + 숫자 -> 문자열로 형변환


스트림
	데이터의 이동 통로
	System.in : 표준 입력 스트림 생성 ( 요청 )
	System.out : 표준  출력 스트림 생성 ( 요청 )

	Buffer : 임시 저장 공간
