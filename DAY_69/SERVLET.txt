서블릿 ( SERVLET )
	서블릿은 Java로 작성된 서버 사이드 컴포넌트로,
	주로 웹 애플리케이션에서 클라이언트의 요청을 처리하고 응답을 생성하는 역할을 합니다.
	서블릿은 자바 클래스이며, 웹 서버(서블릿 컨테이너)에 의해 관리되고 실행됩니다.
	서블릿은 서블릿 컨테이너(예: Apache Tomcat, Jetty)에 의해 실행되며,
	클라이언트의 요청을 받고, 응답을 작성하는 서버 사이드 프로그램입니다.
	기본적으로 HTML을 동적으로 생성하거나,
	클라이언트에게 필요한 데이터를 처리하여 반환하는 역할을 합니다.

서블릿의 역할
	HTTP 요청 처리
		클라이언트로부터의 HTTP 요청을 처리합니다.
	HTTP 응답 생성
		요청에 대한 응답을 생성하고, 클라이언트에게 응답을 반환합니다.
	동적 웹 페이지 생성
		요청에 따라 동적으로 HTML, JSON, XML 등을 생성하여 클라이언트에게 전달합니다.
	데이터베이스와의 통신
		웹 애플리케이션 내에서 데이터베이스와 연동하여 데이터를 처리하고 반환하는 역할을 합니다.


Servlet Filter
	Servlet Filter는 웹 애플리케이션으로 들어오는 요청(request) 또는 응답(response)을
	가로채서 전/후처리를 수행하는 컴포넌트입니다.
