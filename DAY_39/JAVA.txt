클래스 자료형
	클래스 자료형으로 만든 변수는 '참조변수'라고 하고 참조변수는 데이터가
	저장된 위치정보 ( 메모리 주소값 )가 저장된다.


상수
	리터럴 상수 : 이름부여 x, 상수 Pool에 저장, 단순한 수치/값
	심볼릭 상수 : 이름부여 o, final 예약어 사용

	리터럴 접미사 : 리터럴 상수가 저장되는 자료형을 지정
	l, L : long 자료형
	f, F : float 자료형
