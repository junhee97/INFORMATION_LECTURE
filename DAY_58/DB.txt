교차 조인(CROSS JOIN)
	두 테이블의 모든 행의 조합을 반환
	조인 조건이 없으며, 결과는 두 테이블의 행 수를 곱한 만큼의 행을 생성

자체 조인(SELF JOIN)
	하나의 테이블을 자기 자신과 조인하는 방식
	주로 계층적 데이터 구조를 표현할 때 사용


ANSI JOIN
	SQL-92 표준에서 정의된 표준 SQL 조인 방식으로, 테이블 간의 관계를 
	FROM 절과 JOIN 절에서 명시적으로 표현하는 방식입니다.


NATURAL JOIN
	두 테이블에서 동일한 이름을 가진 모든 컬럼에 대해 자동으로 동등 조인을 수행합니다. 
	추가적인 조건을 지정하지 않아도 되며, 공통 컬럼이 여러 개일 경우 모두 조인 조건으로 사용됩니다.


USING 절을 이용한 조인
	두 테이블 간에 동일한 이름의 컬럼을 지정하여 조인을 수행합니다
	조인 조건으로 사용할 컬럼을 명시적으로 지정할 수 있습니다
	USING 절을 사용할 경우, 해당 컬럼에는 별칭을 사용할 수 없으며
	일부 데이터베이스 시스템에서는 지원하지 않을 수 있습니다


서브 쿼리
	SQL 문장 내에 포함된 또 다른 SQL 문장으로 메인 쿼리가 서브 쿼리를 포함하는 종속적인 관계입니다
	괄호 () 안에 작성합니다
	단일 행 또는 복수 행 비교 연산자와 함께 사용됩니다
	서브 쿼리는 메인 쿼리가 실행되기 전에 한 번만 실행됩니다
	ORDER BY는 서브 쿼리에서 사용할 수 없습니다


UNION
	두 SELECT 결과를 합쳐서 하나의 결과로 출력, 중복된 행은 제거 / 정렬 기준은 첫 번째 SELECT문 기준
